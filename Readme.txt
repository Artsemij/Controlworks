В этом примере, класс Note представляет отдельную заметку с заголовком (title) 
и содержимым (content).
Класс NotesManager содержит функционал для работы с заметками. 
Он предоставляет методы create_note для создания заметки, read_notes для чтения списка заметок,
update_note для редактирования заметки и delete_note для удаления заметки. 
Данные о заметках сохраняются в файле notes.json, который хранит список словарей, 
представляющих каждую заметку.
Пример использования показывает, как создавать, читать, редактировать и 
удалять заметки с использованием NotesManager.
